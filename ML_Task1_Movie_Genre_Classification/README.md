Movie Genre Classification using Machine Learning

 Project Overview
This project builds a machine learning model that predicts the genre of a movie
based on its plot summary. It is a text classification problem solved using
Natural Language Processing (NLP) techniques.

This project is completed as Task 1 of the CodSoft Machine Learning Internship.

---

 Dataset
Source: IMDb Genre Classification Dataset (Kaggle)

Dataset Link:
https://www.kaggle.com/datasets/hijest/genre-classification-dataset-imdb

The dataset contains movie plot summaries and their corresponding genres.
